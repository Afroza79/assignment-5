Important FAQ

What is the Document Object Model (DOM)?

DOM is Document Object Model where document object have child nodes like html, body, and other HTML tags. JavaScript can access and modify the DOM to change the content, structure, or style of a webpage dynamically without needing to reload the page.

How do you select an element from the DOM?

There are several methods in JavaScript its depend on the requirements like id, class, tag name, css selector. Examples: Selecting by ID: getElementById(), Selecting by Class: getElementsByClassName(), Selecting by Tag Name: getElementsByTagName()

What is event delegation in the context of the DOM, and why is it useful?

Event delegation attaching event listeners to multiple child elements, you attach a single event listener to a parent element.event delegation process through three phases like Capturing phase, Target phase, Bubbling phase.

How do you manipulate an element's attributes and styles using the DOM?

Using DOM various JavaScript methods and properties.

- Using the classList property adding/removing classes that are defined in an external CSS file like add(), remove(), toggle().
- Also manipulate the text content or HTML content of an element using properties like innerHTML, textContent, and innerText.
